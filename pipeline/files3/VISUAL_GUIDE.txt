IF SOMETHING GOES WRONG:
════════════════════════

❌ Terminal says "Timeout: No response"
   → Browser page is not open or wrong URL
   → Open http://localhost:8080/minimal and try again

❌ Terminal says "Connection failed"  
   → Server is not running
   → Check Terminal 1, restart ./ws_server.js

❌ Browser shows nothing
   → Press F12, check Console tab for errors
   → Refresh page (F5) and try again

❌ Canvas is black (not red)
   → The code is wrong (but test should work)
   → This shouldn't happen with the echo test
